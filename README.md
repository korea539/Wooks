데이터란? / 출처([https://ko.wikipedia.org/wiki/자료](https://ko.wikipedia.org/wiki/%EC%9E%90%EB%A3%8C))

- 데이터가 무엇인가? 라는 질문을 받았다면, 여러분은 어떤 생각을 떠올리십니까? ”영화 매트릭스에 나오는 숫자 정보?”, “외국 드라마에서 보던 해커들의 컴퓨터 화면?” 평소에 데이터와 컴퓨터 공학과 같은 지식에 거리를 두고 살아오셨다면 데이터가 무엇인가라는 질문은 조금 어렵게 들리실 수 있습니다. 하지만 어렵다고 느낀다면 친해지기가 어렵습니다. 데이터의 사전적 의미는 “문자, 숫자, 소리, 그림, 영상, 단어 등의 형태로 된 의미 단위”입니다.
이 사전적 의미를 좀 더 쉽게 접근해봅시다. 우리가 버스를 타고 학교에 가야할 때, 어떤 버스를 타고 가야할 지 검색을 해본다고 가정해봅시다. 핸드폰이나 또는 인터넷을 접속해서 “OO 고등학교 가는 버스”를 검색했습니다. 이 때, 여러분이 검색한 “OO 고등학교 가는 버스”는 한글이라는 문자, 즉 데이터라는 것이죠. 또한 검색을 해서 나온 모든 자료들 또한 데이터입니다. 꼭 글자만 데이터일까요? 우리가 친구와 전화를 하는 것, 뉴진스의 신곡을 듣는 것, 쿠팡에서 물품을 구매하는 과정들 모두 데이터로 이루어져 있습니다.

데이터 분석이란? / 출처([https://ko.wikipedia.org/wiki/데이터_분석](https://ko.wikipedia.org/wiki/%EB%8D%B0%EC%9D%B4%ED%84%B0_%EB%B6%84%EC%84%9D))

- 데이터라는 것에 마음을 조금 열었다면 “데이터 분석”이라는 단어는 어떻게 느껴지십니까?
”분석”이라는 단어에서 거부감을 느끼시지는 않았나요? 하지만 데이터와 마찬가지로 똑같이 생각해봅시다. 데이터 분석의 사전적 정의는 “유용한 정보를 발굴하고 결론적인 내용을 알리며 의사결정을 지원하는 것을 목표로 데이터를 정리, 변환, 모델링하는 과정” 입니다. 데이터의 사전적 의미보다 더 어렵게 느껴집니다.
하지만 좀 더 쉽게 생각해봅시다. ”유용한 정보를 발굴하고 결론적인 내용을 알리며” 라는 뜻은 여러분이 데이터를 통해서 어떤 마트의 야채가 저렴한지, 어떤 카페가 아인슈페너를 맛있게 한다든지, 어떤 영화가 평점이 좋다든지와 같은 정보를 찾고 알리는 것을 말합니다. 조금만 생각해보면 우리가 일상에서 많이 해왔던 행동들입니다. “의사결정을 지원하는 것을 목표로 데이터를 정리, 변환, 모델링하는 과정” 라는 뜻은 여러분들이 찾아낸 데이터들을 정리하고 고민하여, 판단을 하는데 도움을 주는 것이 목표라는 의미입니다. 사진이 잘 나오는 카페를 검색해보고 가격과 거리를 감안하여 한 곳을 결정하거나, 친구들의 추천을 받은 명소들 중 3시간 안에 다녀올 수 있는 곳을 선택하여 가보는 것이 데이터 분석의 과정으로 볼 수 있습니다.

데이터 분석의 흐름 / 출처([https://rstudio-pubs-static.s3.amazonaws.com/382545_098d268806f449c496734236e0b97493.html](https://rstudio-pubs-static.s3.amazonaws.com/382545_098d268806f449c496734236e0b97493.html))

- 그렇다면 데이터 분석은 어떻게 이루어질까요?
데이터 분석의 흐름은 크게 3가지로 나타납니다.
1. 데이터 수집 2. 데이터 전처리 3. 데이터 시각화
첫 번째 데이터 수집은 지금까지 해온 얘기들과 동일합니다. 여러분들이 좀 더 좋은 효율을 볼 수 있도록 데이터를 모으는 과정을 데이터 수집이라고 합니다.
두 번째 데이터 전처리는 처음 들어보신 분들도 계실겁니다. 하지만 전혀 어렵지 않은 의미입니다. 데이터 전처리는 여러분들이 가져온 데이터들을 걸러내서 사용하겠다라는 의미입니다. 우리가 10군데의 맛집을 찾아놨다고 가정합시다. 오늘 그 10군데 중 어디를 가야할 지 고민을 하게됐습니다. 두 곳은 오늘 휴업을 하셔서 제외를 하였고, 다른 세 곳은 줄이 너무 길어 제외를 하였습니다. 이처럼 여러분이 수집한 데이터에서 필요에 의한 데이터들만 가공하는 것을 전처리라고 합니다.
마지막 데이터 시각화는 데이터를 더 쉽게 이해할 수 있도록 그래픽으로 표시하는 것을 말합니다. 다들 주식 그래프는 한 번씩 보셨을겁니다. 주식그래프를 보고 있으면 해당 종목의 주가가 어떻게 변해왔는지를 어렵지 않게 이해할 수 있습니다. 하지만 이 그래프를 하나하나의 데이터로 본다면 어떨까요?
