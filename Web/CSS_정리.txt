# CSS 정리


<style>
@@@
</style> = CSS를 사용하겠다 라는 의미

; = 세미콜론은 하나의 동작(작업)이 끝났다는 표시이므로 끝날때마다 표시해주기

head에 있는 a는 웹페이지의 모든 태그를 선택한다는 Selector
color:red; 와 같은 태그들에 대해서 어떤 효과를 줄지에 대한 선언(Declaration) / 선택자
color는 Property(속성), red는 Value(값)이라고 부른다.

강한 순서는 id(표현은 #id)가 가장 우선 다음 class(표현은 .class) 다음 tag(예시로 a)

id는 유일무이의 값

css selector 를 검색해서 다양한 셀렉터(선택자)들을 볼 수 있다.

block level = 홈페이지 한 단을 다 먹는 레벨
inline level = 단어 하나만을 먹는 레벨

div = block level
span = inline level
grid columns의 용어 1fr = 화면을 분할해서 계속 반영해주는 기능

caniuse 사이트 = css에서 어떤 선택자를 쓸 수 있을지 나오는 사이트

mediaquary = 화면 사이즈에 맞춰 출력할지 안할지를 해주는 기능

cache = 저장하다.

프로그램(program)이란 순차적인 기능을 짜는 것을 의미하고 있다.
HTML은 시간 순서가 아니라 단순히 보여주는 언어여서 프로그래밍 언어가 아니지만
JS는 사용자와 상호작용하는 언어기 때문에 프로그래밍 언어라고 부를 수 있다.
